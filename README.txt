一、YAML基本介绍
	1 yaml 定义和作用
		yaml 就是一种可以支持多种编程语言的配置文件书写语言，我们学习 yaml 的目的就是为了将我们PO自动化测试脚本中的数据抽离出来当前的逻辑
		而形成可以单独管理的数据文件。
		
	2 yaml 基本语法规则
		01 yaml 用缩进来表示层级关系	【这里的缩进必须是空格，在使用pycharm时可以自动转化】
		02 yaml 同级的数据左侧必须对齐
		03 yaml 数据文件后缀可以是 yaml 或者 yml
		04 yaml 文件可以用来存放不同的数据类型
		
	3 yaml 数据类型
		01 对象：python中的字典
		02 数组：python中的列表 和 元组
		03 纯量：字符串	布尔值 整数 浮点数	None
	4 yaml 文件书写细节
		01 存放字典的时候需要采用 键值对 的形式，键名: 键值【冒号后面需要有空格】
		02 存放列表数据时采用 - 值 形式来表示，此时 - 后面也要添加空格
